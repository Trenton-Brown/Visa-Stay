# Visa Stay

A modern, minimal visa tracking app for digital nomads and slow travelers.

## Features

- **195+ Countries**: Comprehensive coverage of visa requirements worldwide
- **Real-time Data**: Always up-to-date information from official sources
- **Trip Planning**: Plan your multi-country journeys with confidence

## Tech Stack

- React 18 + TypeScript
- Vite
- shadcn/ui components
- Tailwind CSS

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

```
src/
  ├── components/
  │   └── ui/          # shadcn/ui components
  ├── lib/
  │   └── utils.ts     # Utility functions
  ├── App.tsx          # Main app component
  ├── main.tsx         # Entry point
  └── index.css        # Global styles
```
